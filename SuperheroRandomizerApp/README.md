# Superhero Randomizer iOS App

A clean, animated iOS application that fetches and displays a random superhero from the public Superhero API every time you tap the button.

## Features

- One-time download of the full superhero database on app launch  
- “Randomize” button – tap to get a new random superhero instantly  
- Displays 10+ superhero attributes
- Large hero image with rounded corners, subtle shadow, and smooth cross-dissolve animation  
- Loading spinner + “Loading…” placeholders while fetching data and image  
- Proper error handling for network failures and JSON parsing  
- Beautiful fade/cross-dissolve transitions when a new hero appears  
- 100% UIKit, no third-party dependencies
