# FavoritesManager

Simple UIKit app (Storyboard) with TabBar and 4 categories: Movies, Music, Books, Courses.

## How to run
- Open FavoritesManagerApp.xcodeproj
- Build & Run on iOS 15+ simulators

## Features
- TabBar with 4 tabs
- UITableView with custom cells (image + title)
- Detail view with image, title, description and review
- All layouts implemented with Auto Layout (Storyboard)
